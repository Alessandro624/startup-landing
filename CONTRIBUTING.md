# Contributing

Thanks for improving this template.

## Setup

1. npm install
2. cp .env.example .env.local
3. npm run dev

## Guidelines

- Keep copy and assets in data/content.ts and public/
- Keep placeholder tokens in comments for search
- Use ASCII only in source files
- Run npm run lint and npm run type-check before a PR

## Pull requests

- Describe what changed and why
- Include screenshots for UI changes
